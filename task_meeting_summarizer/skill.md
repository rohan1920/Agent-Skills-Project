# Task & Meeting Summarizer

## Purpose
Summarizes meeting notes or task lists into actionable points to save time.

## Example
Input:
"Discussed new features for the Rawaj project, need to update the homepage UI, fix the login bug, assign tasks to team."

Output:
- Update homepage UI
- Fix login bug
- Assign tasks to team

## How it Replaces Your Job
Instead of manually reading notes and creating task lists, this skill automates the summarization.
