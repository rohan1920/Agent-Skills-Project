# Syllabus Scraper Skill

## Purpose:
Automatically fetch syllabus or course content from GitHub repositories and summarize it in an organized format.

## What it replaces:
Manually checking repositories and organizing lessons/topics.

## How it works:
1. Input: GitHub repository URL + course/folder name
2. Process: Read repository structure, collect lesson names and sections
3. Output: Structured syllabus with topics and assignments for easy review
