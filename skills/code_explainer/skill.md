# Code Explainer Skill

## Purpose:
Explain any code snippet in plain English, line by line, so beginners can understand programming logic quickly.

## What it replaces:
Manual explanation of code for students or colleagues.

## How it works:
1. Input: Any code snippet (Python, JavaScript, or any language)
2. Process: Read the code line by line, identify loops, conditions, variables, and functions
3. Output: Clear, concise explanation of each line, describing what the code does in simple words
