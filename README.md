# Agent Skills Project

This project contains **5 reusable agent skills** that automate parts of my daily work.  
Each skill is designed to perform high-value, repetitive tasks efficiently, replacing manual effort and saving time.


## Skills Overview

1. **Code Explainer**
   - Explains code snippets in simple terms.
   - Replaces the task of manually understanding and documenting code.
   - [See Skill Details](skills/code_explainer/skill.md)

2. **Syllabus Scraper**
   - Extracts and organizes syllabus data from GitHub repositories.
   - Replaces the manual process of collecting study material.
   - [See Skill Details](skills/syllabus_scraper/skill.md)

3. **Medical Advice Summarizer**
   - Summarizes medical advice from sources into easy-to-read points.
   - Replaces the task of reading long articles or documents.
   - [See Skill Details](skills/medical_advice_summarizer/skill.md)

4. **Social Media Post Generator**
   - Generates posts for social media platforms.
   - Replaces the repetitive task of writing promotional content manually.
   - [See Skill Details](skills/social_media_post_generator/skill.md)

5. **Task & Meeting Summarizer**
   - Summarizes meeting notes or task lists into actionable points.
   - Replaces the manual effort of reviewing notes and creating task lists.
   - [See Skill Details](skills/task_meeting_summarizer/skill.md)



## How to Use

- Each skill has its own folder under `skills/`.
- Open the `skill.md` file inside each folder to see:
  - Purpose
  - Example input/output
  - How it replaces a part of your job
- Clone this repo to explore and use these skills.
